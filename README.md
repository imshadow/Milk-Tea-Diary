这里记录的是奶茶, 亦是生活中的那些甜蜜与美好.

## 介绍

学习 Next.js, 根据官方教程 [https://nextjs.org/learn/dashboard-app](https://nextjs.org/learn/dashboard-app) 完成了教程里的官方项目

项目：[https://github.com/vercel/next-learn](https://github.com/vercel/next-learn)

然后在原官方项目的基础上进行改造

改造成当前的**奶茶日记**
